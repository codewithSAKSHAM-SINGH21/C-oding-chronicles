Git commands :


if you already have a GitHub repo you just have to copy its repo link and the in vs code terminal :
write :
git clone <link>
git add .
git commit -m "any message you want to type"
git push origin <branch name>

 
AND IF YOU ARE MAKING A NEW REPO :
COMMAMNDS ARE ALREADY IN THE NEW REPO TO INSERT :

git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/codewithSAKSHAM-SINGH21/HEY.git
git push -u origin main
